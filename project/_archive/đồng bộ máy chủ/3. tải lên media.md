# 3. Tải Lên Media (Media Upload)

## Trạng Thái: ✅ ĐÃ HOÀN THÀNH (PASSED)

## Mô Tả
Cơ chế upload các file media (ảnh sản phẩm, ảnh CCCD) lên server lưu trữ (S3/Local Storage) thông qua API Multipart Upload riêng biệt.

## Chi Tiết Kỹ Thuật

### 1. Endpoint
- **URL:** `POST https://quanly.hoadigital.com/api/upload`
- **Content-Type:** `multipart/form-data`

### 2. Cấu Trúc Dữ Liệu
Form Data gửi lên:
- `file`: (Binary data của ảnh)
- **Validation:** Server chỉ chấp nhận file ảnh (binary), từ chối text file.
- `type`: `product_image` hoặc `customer_cccd`
- `license_key`: (Để xác thực quyền upload)

### 3. Phản Hồi Từ Server (Response)
```json
{
  "success": true,
  "url": "https://quanly.hoadigital.com/storage/uploads/product_image/..."
}
```
