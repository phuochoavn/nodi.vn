# 4. Cập Nhật Phần Mềm (App Update / OTA)

## Trạng Thái: ✅ ĐÃ HOÀN THÀNH (PASSED)

## Mô Tả
Tính năng tự động kiểm tra phiên bản mới mỗi khi khởi động ứng dụng và nhắc người dùng cập nhật.

## Chi Tiết Kỹ Thuật

### 1. Endpoint
- **URL:** `POST https://quanly.hoadigital.com/api/check-update`
- **Method:** `POST`

### 2. Cấu Trúc Dữ Liệu (Payload)
Client gửi lên:
```json
{
  "current_version": "1.0.0",
  "hwid": "..."
}
```

### 3. Phản Hồi Từ Server (Response)
```json
{
  "latest_version": "1.1.0",
  "has_update": true,
  "download_url": "https://quanly.hoadigital.com/downloads/AgriPOS_Setup_1.1.0.exe",
  "release_notes": "- Thêm tính năng đồng bộ ảnh\n- Sửa lỗi in hóa đơn"
}
```
