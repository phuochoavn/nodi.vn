### 3.3. 🟪 MISA — "Hệ điều hành doanh nghiệp VN"

> **Chi tiết**: [misa.md](file:///d:/App%20pc/New%20folder/agri-pos-tauri/project/chien-luoc/doi-thu/misa.md)

| Mục | Tóm tắt |
|---|---|
| **Vị thế** | #1 kế toán VN, 250K+ DN, 30 năm |
| **Chiến lược** | Chiếm kế toán → cross-sell toàn bộ |
| **VTNN** | eShop phiên bản VTNN riêng |
| **Điểm mạnh** | Lock-in qua hệ sinh thái 7 module, MISA Academy, bán an toàn pháp lý |
| **Điểm yếu** | Learning curve cao, chậm giờ cao điểm, F&B yếu |
| **Mở rộng** | MISA AVA (AI voice), MISA Lending (fintech) |

**5 bài học vĩ đại từ MISA:**
1. **Chiếm "trái tim" hệ thống** (kế toán) → mở rộng
2. **Chi phí chuyển đổi vô hình** → 7 module liên kết = không thể rời
3. **Đào tạo = bán hàng** → đưa vào ĐH, triệu SV "chỉ biết MISA"
4. **Bán an toàn pháp lý** → "không bị phạt" thay vì "tính năng tốt"
5. **Biến KH thành sales** → ASP kế toán giới thiệu = điểm thưởng
