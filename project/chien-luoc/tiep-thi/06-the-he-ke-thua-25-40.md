## 4. Thiết Kế Sản Phẩm — 1 Database, 2 Giao Diện

### Simple Mode (F1 — Desktop)

| Nguyên tắc | Chi tiết |
|-----------|---------|
| 🔤 Phông chữ ≥ 16px | Tương phản cao (đen/trắng/vàng nhạt) |
| 🚫 Không icon ẩn | Không hamburger menu → Nút lớn: "BÁN HÀNG", "GHI NỢ", "IN HÓA ĐƠN" |
| 📶 Offline-first | Chấm xanh: "Lưu trữ ngoại tuyến an toàn" |
| 📒 Sổ nợ trực quan | Mô phỏng sổ vật lý, tìm theo biệt danh ("Tư Lúa", "Bảy Sầu Riêng") |
| 🖨️ Plug-and-play | Máy in nhiệt + barcode scanner |
| 🚫 **ẨN**: Analytics, cấu hình, phân quyền | Tránh thao tác nhầm → hoang mang |

### Full Mode (F2 — Mobile)

| Tính năng | Chi tiết |
|-----------|---------|
| 🤖 **AI Chatbot** | Chẩn đoán bệnh camera → Kê thuốc từ kho |
| 📊 Dashboard | Dòng tiền, cảnh báo hàng cận date, nợ xấu |
| ⚙️ Setup mọi thứ | Quét barcode nhập kho, chụp ảnh SP, cấu hình giá sỉ/lẻ |
| ☁️ Cloud-synced | Cập nhật real-time: thời tiết, giá, AI |

### Bảng so sánh

| Tiêu chí | Simple Mode (F1) | Full Mode (F2) |
|----------|:----------------:|:--------------:|
| Thiết bị | Desktop Windows | Smartphone |
| UI | Geriatric, sổ vật lý | Modern Mobile UX |
| Mạng | **Offline-first** | Cloud-synced |
| Core | Ghi nợ, in bill, quét barcode | AI, analytics, quản lý kho |
| Ẩn | Charts, config, permissions | In hóa đơn, POS 16:9 |
